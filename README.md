# InterviewPractice

My personal interview preparation workspace — code, notes, and practice across multiple topics. Tracked here so I can learn from anywhere and follow my progress over time.

## Structure

| Folder | What's inside |
|--------|---------------|
| **DSA** | Data Structures & Algorithms — trees, graphs, DP, recursion, hashing, tries, heaps, segment/Fenwick trees, string algorithms, LeetCode practice, and more |
| **JavaLearning** | Core Java — OOP, collections, multithreading, file handling, new Java features, Java basics |
| **SpringBoot** | Spring Boot — JPA, Spring Web, Annotations, Microservices, Kafka, Spring AI, React |
| **LLD** | Low-Level Design — SOLID, design patterns (Observer, Adapter, Composite, Singleton, Command, Builder, Factory, Strategy), exercises |
| **AWS** | AWS — IAM, VPC, General SDK |
| **AIML** | AI / Machine Learning |
| **CICD** | CI/CD practice |

## How I use this

Every study session:

```bash
git add .
git commit -m "describe what I learned / changed"
git push
```

Every commit is a timestamped snapshot of my progress.
